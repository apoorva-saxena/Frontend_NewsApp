# NewsApp Frontend

An under development app that helps you organise your news resources.

## Technologies used

#### Front End
- React

#### Back end
- Node
- Express

(Find the backend code in a repository: Backend-NewsApp)

used BBC news api for fetching the various resources and required data

## Setup

Just run :
```
$ yarn start

```
